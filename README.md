# README
## 開発言語
* Ruby 3.3.0
* Ruby on Rails 7.1.5.1
* PostgreSQL 16.9
## 実行手順
```
$ git clone https://github.com/moaoyama/org_app_tottoku.git
$ cd tottoku
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails server
```
